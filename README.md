# 📈 Stock Market Price Predictor

A deep learning web app that predicts stock closing prices using a stacked LSTM neural network, trained on historical market data and deployed with an interactive Streamlit interface.

![Streamlit](https://img.shields.io/badge/Streamlit-App-red)

---

## 🔍 Overview

This project uses a multi-layer **LSTM (Long Short-Term Memory)** network to forecast stock prices based on historical closing price trends. Users can input any stock ticker, and the app fetches live historical data, visualizes moving averages, and displays predicted vs. actual price trends.

---

## ✨ Features

- **Live data fetching** — pulls historical stock data directly from Yahoo Finance via `yfinance`
- **Moving average visualizations** — 50-day, 100-day, and 200-day moving averages plotted against actual closing prices
- **LSTM-based price prediction** — a 4-layer stacked LSTM model trained on 10 years of historical data
- **Interactive web interface** — built with Streamlit for real-time ticker input and visualization
- **Predicted vs. actual comparison** — visual chart comparing the model's forecasts against real prices

---

## 🧠 Model Architecture

The model is a sequential stacked LSTM network built with Keras:

| Layer | Units | Activation | Dropout |
|-------|-------|------------|---------|
| LSTM 1 | 50 | ReLU | 0.2 |
| LSTM 2 | 60 | ReLU | 0.3 |
| LSTM 3 | 80 | ReLU | 0.4 |
| LSTM 4 | 120 | ReLU | 0.5 |
| Dense (Output) | 1 | — | — |

- **Input:** 100-day sliding window of scaled closing prices
- **Optimizer:** Adam
- **Loss function:** Mean Squared Error (MSE)
- **Training data:** 10 years of historical closing prices (2012–2022), scaled with `MinMaxScaler`
- **Train/test split:** 80/20

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- pip

### Installation

```bash
# Clone the repository
git clone https://github.com/ananth-00/<your-repo-name>.git
cd <your-repo-name>

# Install dependencies
pip install -r requirements.txt
```

### Run the app

```bash
streamlit run app.py
```

Then open the local URL shown in your terminal (typically `http://localhost:8501`) and enter any stock ticker symbol (e.g., `GOOG`, `TSLA`, `AAPL`) to see predictions.

---

## 📊 How It Works

1. User enters a stock ticker in the web interface.
2. Historical price data is fetched via the Yahoo Finance API.
3. Data is preprocessed and scaled using the same pipeline used during training.
4. The trained LSTM model generates predictions on a rolling 100-day window.
5. Results are visualized alongside actual prices and moving averages.
